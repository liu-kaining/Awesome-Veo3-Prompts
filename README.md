## 🎯 使用说明

每个提示词文件都包含：
- 完整的英文/JSON提示词
- 中文翻译版本
- 详细的场景描述和参数设置

## 📝 格式说明

提示词支持两种格式：
1. **JSON格式**: 包含详细的参数结构（镜头、灯光、音频等）
2. **纯文本格式**: 叙事性描述，适合创意场景
